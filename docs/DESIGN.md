# ReMotion Design Rules

## Brand Meaning
- Blue = stability, control, primary actions
- Green = recovery, progress, success

## Logo Usage
- Never recolor logos
- Use logo-light on light backgrounds
- Use logo-dark on dark backgrounds

## UI Colors
- Primary actions → brand.blue.700
- Secondary / success → brand.green.700
- No gradients in UI elements
- No arbitrary Tailwind colors

## Dark Mode
- Always use semantic tokens
- Never hardcode colors in components
